#  Land Price Prediction

## Project Overview

This project predicts land prices based on distance from NIMS University using Machine Learning.

The project demonstrates:

* CSV and JSON data handling
* Dataset merging
* Data cleaning
* Feature engineering
* Data visualization
* Linear Regression model

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Features

* Multiple dataset integration
* Missing value handling
* Duplicate removal
* Near NIMS / Outside NIMS categorization
* Land price prediction

## Machine Learning Model

Linear Regression was used to predict land prices based on distance.

## Project Workflow

Data Collection → Data Cleaning → Dataset Merging → Feature Engineering → Visualization → Prediction

## Output

The model predicts land prices according to distance from NIMS University.
